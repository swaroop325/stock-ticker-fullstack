# Project Details

This project was built with 
    - React JS as the frontend 
    - Node JS as the backend

## To start the project

## Frontend

Navigate to frontend folder
`cd frontend`

### `npm install` 
To install dependencies

### `npm start`
To start the project

### `npm test`
To run unit tests for the project

## Backend
Navigate to frontend folder
`cd backend`

### `npm install` 
To install dependencies

### `npm start`
To start the project

### `npm test`
To run unit tests for the project
